CloudLogMiner or CLM is a tool under development for analyzing Web Application logs from the very beginning of development.

As such, it focuses on easing log review efforts when locating bugs in a distributed Software System.

User guide can be found in CLM GitHub repository

The tool fetches logs from an elasticSearch instance.

Built with Angular2 and ag-grid.

CloudLogMiner is a Bachelor thesis project by Silvia Moreno Uribarri
